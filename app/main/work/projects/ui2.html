ui2 is a Python module for the iOS app Pythonista Pythonista which builds on top of Pythonista's ui module to add functionality and convenience. ui2 empowers script authors to take advantage of a much wider range of components, allowing them to create more complex, beautiful, and effective interfaces. The project was commissioned by another community member, who thought that I had the necessary skill to carry out the project and paid me to complete it.

ui2 lets users access UI components like MKMapView that are not available through the original ui module, and also provides pure-python abstractions of ui features to add flexibility and convenience, such as allowing the chaining of multiple animations.
